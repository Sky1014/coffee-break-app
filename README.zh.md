# Coffee Break ☕

**跨平台休息提醒应用，帮你定时离开屏幕。**

专注工作 30 分钟，休息 60 秒——站起来、伸展身体、回来时焕然一新。

**[下载 Windows 便携版](https://github.com/Sky1014/coffee-break-app/releases/latest)** — 无需安装，解压即用。

---

## ✨ 功能特性

### 桌面端 (Windows)
- **设置窗口** — 自定义工作/休息时长、语言等
- **系统托盘运行** — 安静地在后台运行，关闭窗口最小化到托盘
- **全屏休息弹窗** — 不会错过
- **自定义时长** — 工作时长 5-120 分钟，休息时长 0.5-10 分钟
- **勿扰模式** — 通过滑块或托盘菜单暂停提醒 5-480 分钟
- **自定义音效** — 可选择本地音频文件作为休息音效（支持 ogg/mp3/wav/m4a/aac/flac）
- **应用内更新** — 检查 GitHub 最新版本，一键下载并应用更新
- **自动暂停/恢复** — 检测屏幕锁屏状态并自动调整
- **程序化音效** — 合成咖啡冲泡音效（可静音）
- **CSS 动画** — 流畅的进度环、咖啡杯和蒸汽效果
- **中英双语** — 自动检测系统语言，支持手动切换，托盘菜单完整翻译
- **开机自启** — 可选，在设置中配置
- **便携版 + 安装包** — 解压即用的 ZIP，或带桌面快捷方式的 NSIS 安装程序

### iOS / watchOS（开发中）
移动端和手表端原型正在开发中，核心逻辑和 SwiftUI 骨架已就位，完整的 iOS/watchOS 应用尚未完成。

## 🚀 快速开始（桌面端）

**方式一 — 安装程序（推荐）：** 从 [Releases 页面](https://github.com/Sky1014/coffee-break-app/releases/latest) 下载 `CoffeeBreak-x.x.x-Setup.exe`，运行并按照向导操作。桌面快捷方式和开始菜单项会自动创建。

**方式二 — 便携版：** 下载 ZIP 文件，解压到任意位置，运行 `CoffeeBreak.exe`。无需安装。

**系统要求：** Windows 10/11 x64

如果你想从源码构建，需要 Node.js 18+。桌面应用使用 Vite + Electron，安装依赖后即可本地运行。

## 📋 休息模式

桌面应用支持通过设置窗口完全自定义间隔时长。

| 模式 | 间隔 | 状态 |
|------|------|------|
| 自定义工作 | 5-120 分钟（默认 30 分钟） | ✅ 可用 |
| 自定义休息 | 0.5-10 分钟（默认 1 分钟） | ✅ 可用 |
| 勿扰模式 | 5-480 分钟 | ✅ 可用 |
